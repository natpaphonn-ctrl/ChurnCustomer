Methodology
Analysis Pipeline

Data Loading

โหลด 296,351 rows, 74 columns, จัดการ missing values และ duplicate columns

EDA

วิเคราะห์ demographics, tenure, purchase history และ trend รายงวด

Feature Engineering

สร้าง 13 features: tenure, purchase ratio, trend, streak, max gap ฯลฯ

Preprocessing

Clip outliers 99th pct, RobustScaler, ตัด features ที่ corr > 0.8 เหลือ 11 ตัว

K-Means Clustering

ทดสอบ K=2-12, เลือก K=8 เพื่อเห็นกลุ่มย่อยที่น่าสนใจ

Cluster Profiling

Radar chart, bar chart, purchase array — ตั้งชื่อและวิเคราะห์แต่ละกลุ่ม
